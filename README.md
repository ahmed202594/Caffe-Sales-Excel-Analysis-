# Caffe-Sales-Excel-Analysis-
End-to-end sales analysis of a Coffee Shop over 15 months (2024–Q1 2025). Utilizing advanced Excel features—including Pivot Tables, Dynamic Charts, Slicers, and formulas like XLOOKUP/SUMIFS—to analyze 39k+ records, revealing a 43% growth rate in 2025 and providing data-driven recommendations for inventory and seasonal marketing.
